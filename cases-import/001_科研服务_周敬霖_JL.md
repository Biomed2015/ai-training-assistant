# 案例编号：001
# 行业：科研服务
# 客户类型：SCI论文内审

## 客户背景
- **姓名**：周敬霖 (JL)
- **研究方向**：病理联合影像+空转+单细胞
- **目标期刊**：15分左右SCI期刊
- **心仪期刊**：npj digital medicine、Redox Biology

## 客户需求
1. 指定期刊内审服务
2. 希望文章能顺利发表在一区期刊
3. 担心被拒稿后怎么办
4. 需要发票报销（试剂或服务）

## 沟通过程
### 阶段1：初次接触（2025-09-12）
- 客户主动咨询npj digital medicine期刊
- 询问是否有对接编辑
- 表达对该期刊的偏好（感觉很多文章都是找人的）
- 通过微信小程序看到服务

### 阶段2：需求确认
- 销售明确告知按中科院分区定价
- 发服务流程和费用表（500元诚意金 + 5000元文稿服务费 + 渠道费）
- 客户询问万一被拒怎么办
- 销售解释会做初审和期刊匹配，确保期刊可投

### 阶段3：方案洽谈
- 确认意向期刊：Redox Biology（一区，IF=11.9）
- 文稿服务费5000元（诚意金抵扣后）
- 告知可以开发票（试剂耗材或技术服务）

### 阶段4：疑虑处理
- 客户担心"是否有熟人"
- 销售强调根据文章具体匹配，不是空口承诺
- 解释做这一行是协助文章录用才能收到渠道费

## 成交方案
- **期刊**：Redox Biology（中科院1区，IF=11.9）
- **服务**：SCI期刊内审服务
- **费用**：文稿服务费5000元 + 渠道费（录用后结算）
- **付款方式**：银行转账，可开发票

## 成功原因
1. 专业的期刊匹配分析报告建立信任
2. 坦诚沟通，不夸大承诺
3. 付款流程正规（有银行账号、正规发票）
4. 强调"作者自行投稿"区别于代理黑箱操作
5. 合理的退款保障（渠道费规则）

## 可复用的营销话术
1. **区别代理**："我们绝对不合作代理，我们只对接个人客户，拒绝中间商赚差价"
2. **建立信任**："你自己邮箱投稿，收到通知可以直接看"
3. **处理疑虑**："期刊匹配表里面的期刊都是可以投的，不然不会给你匹配过来"
4. **解释费用**："做这一行都是协助文章能够收到修改和录用，你才给我们渠道费"
5. **老客户优惠**："老客户不会直接给新价格，会有优惠空间"
