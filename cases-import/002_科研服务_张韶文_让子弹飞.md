# 案例编号：002
# 行业：科研服务
# 客户类型：SCI翻译润色+内审

## 客户背景
- **姓名**：张韶文（让子弹飞一会）
- **身份**：青海麻醉专硕研究生，明年毕业
- **紧急原因**：医院伦理过审晚，中华投稿因伦理问题被拒，想申博

## 客户需求
1. 将中文文章翻译成英文并润色
2. 选合适期刊投稿
3. 周期要短，年底前录用（4个月）
4. 目标：三区即可

## 沟通过程
### 阶段1：初次咨询（2025-07-29）
- 客户主动咨询翻译润色服务
- 想了解费用和周期
- 询问是否方便语音

### 阶段2：需求挖掘
- 销售坚持文字沟通（客户多，文字更清晰）
- 了解客户背景：中文被拒、时间紧迫、需申博
- 评估可行性：4个月三区有机会

### 阶段3：方案介绍
- 介绍一体化服务包含：翻译、润色、排版、期刊推荐、Cover letter
- 解释SCI审稿周期：比国内短，但无法打包票
- 明确费用：三区服务费约4000元

### 阶段4：需求确认
- 客户想"写发一体化"
- 销售澄清：写发一体化是6万起步（针对没有文章的定制服务）
- 客户现有文章，适合"内审协助"服务

## 成交方案
- **期刊**：BMC Anesthesiology（三区）
- **服务**：SCI期刊内审服务（含翻译润色）
- **费用**：三区定价 + 文稿服务费

## 成功原因
1. 准确识别客户需求：时间紧迫、只需内审协助
2. 没有强行推销高价"写发一体化"
3. 坦诚说明SCI审稿的运气因素
4. 用"明白人"话术建立信任

## 可复用的营销话术
1. **处理时间紧迫**："4个月在国内容不了，SCI周期绝对比国内短"
2. **解释运气因素**："审稿人不是机器，主观能动性占一部分"
3. **坦诚沟通**："没有100%绝对的事情，做实验都有风险"
4. **识别需求**："你这个情况不属于写发一体化，写发是针对没有文章的"
5. **建立信任**："跟明白人说话就是爽快，事情也会办得顺利"
