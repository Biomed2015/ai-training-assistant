# 案例编号：003
# 行业：科研服务
# 客户类型：老客户续费

## 客户背景
- **姓名**：JMJ
- **身份**：老客户，之前合作过
- **新需求**：第二篇文章，继续享受老客户优惠

## 客户需求
1. 安排第二篇文章的内审服务
2. 希望能按旧价格计算（已涨价）
3. 国庆节前希望能投出去
4. 后续还有2篇文章要做

## 沟通过程
### 阶段1：老客户维护（2025-09-01）
- 销售主动联系询问安排
- 提醒客户名额有限，按周划分

### 阶段2：价格调整说明
- 告知客户价格体系已更新，各分区加1万
- 老客户仍按原价体系，但需申请优惠

### 阶段3：争取优惠
- 客户询问自己是否也按旧价格
- 销售承诺帮忙申请，预估25000（二区原价3万）
- 强调老客户不会给新价格

### 阶段4：时间紧迫
- 客户因上课、大动物实验耽搁
- 销售提醒：只剩十来天，国庆前要投需尽快
- 客户承诺今晚发送文章

## 成交方案
- **期刊**：二区
- **服务**：SCI内审服务
- **费用**：25000元（老客户优惠价，原价30000）
- **付款**：银行转账，诚意金抵扣

## 成功原因
1. 老客户关系维护做得好
2. 主动提醒名额有限，制造紧迫感
3. 灵活的价格政策（老客户优惠）
4. 承诺帮忙申请优惠价格

## 可复用的营销话术
1. **名额有限**："我们每个月名额都有限，都是按周来划分的"
2. **老客户优惠**："老客户不会直接给新价格，会有优惠空间"
3. **强调时间**："现在离国庆也就十来天，我们稿件也比较多"
4. **复购引导**："做完实验再联系我就行"
5. **感情牌**："就像熟悉的店一样，涨价也会用原价收你"
