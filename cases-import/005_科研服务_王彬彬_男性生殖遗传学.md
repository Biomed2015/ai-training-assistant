# 案例编号：005
# 行业：科研服务
# 客户类型：SCI内审-保密需求

## 客户背景
- **姓名**：王彬彬
- **研究方向**：男性生殖遗传学研究
- **目标**：4-5分，分区不限
- **截止时间**：明年3月答辩前拿到接收函

## 客户需求
1. 文章一直被拒，需要帮助
2. 4-5分期刊
3. **特殊需求**：不想让导师知道
4. 可以不提供单位和名字

## 沟通过程
### 阶段1：初次咨询
- 通过公众号看到联系方式
- 坦言文章一直被狙
- 询问流程和价格

### 阶段2：需求确认
- 销售解释分区定价和服务流程
- 客户担心时间：明年3月答辩
- 销售给出时间线：2-3个月录用

### 阶段3：保密处理（关键）
- 客户坦言不想让老板知道
- 询问是否可以不发单位和名字
- **销售果断回应**：完全可以，保护隐私

### 阶段4：名额催促
- 销售提醒本月名额所剩无几
- 客户表示马上发送文章

## 成交方案
- **期刊**：待匹配（男性生殖遗传学方向）
- **服务**：SCI内审服务
- **费用**：待定（根据分区）

## 成功原因
1. **关键突破**：理解客户"不想让导师知道"的隐私需求
2. 果断承诺保护隐私，建立深度信任
3. 名额有限制造紧迫感
4. 时间线明确（2-3个月）

## 可复用的营销话术
1. **保护隐私**："你可以把隐私全部去掉，这个不重要，有文章就行"
2. **理解客户**："大家肯定是不想让老板和领导知道的，这个我们完全理解"
3. **制造紧迫**："这个月名额也没剩几个了，应该就是这几天会安排完"
4. **时间承诺**："合作成功的作者也就是2~3个月差不多了"
