# 案例编号：006
# 行业：科研服务
# 客户类型：SCI内审-公共数据库

## 客户背景
- **姓名**：宋延伟
- **研究方向**：公共数据库+湿实验文章
- **目标期刊**：尽快接受，不挑分区，3分即可
- **状态**：已完成初稿，投Journal of translational medicine被拒

## 客户需求
1. 想选择能被快速接受的期刊
2. 想了解内审服务内容
3. 担心录用率问题
4. 时间紧迫（节后要开始）

## 沟通过程
### 阶段1：咨询期刊
- 询问某个杂志是否可操作
- 说明自己写了公共数据库+湿实验文章
- 想选能快速接受的杂志

### 阶段2：了解内审服务
- 询问内审服务是什么内容
- 销售解释初审评估+期刊匹配

### 阶段3：名额说明
- 销售告知本月名额已满，需等到下个月
- 可以先做初审和期刊匹配

### 阶段4：疑虑处理
- 客户坦言：对分区和IF没要求，3分就可以
- 主要担心录用率
- 销售解释：配合好录用率就很高

## 成交方案
- **期刊**：待匹配（客户要求3分即可，快速接受）
- **服务**：SCI内审服务
- **时间**：节后开始，先做初审匹配

## 成功原因
1. 准确把握客户核心需求：不挑分区，只要快
2. 名额有限策略有效
3. 用医疗类比解释配合重要性

## 可复用的营销话术
1. **解释录用率**："配合的好它的录用率就很高，有的作者不配合，那没办法。就跟医院治病一样"
2. **名额有限**："每个月只能配30个名额，多了搞不完"
3. **解释流程**："渠道费是在录用之后才收取的，前期就是一个稿件服务费"
