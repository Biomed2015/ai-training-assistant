# 案例编号：009
# 行业：科研服务
# 客户类型：SCI内审-产科综述

## 客户背景
- **姓名**：诗炆
- **研究方向**：产科合并神经系统疾病（妊娠合并CVST）
- **目标期刊**：一区，越快越好
- **截止时间**：1月份前录用（用于申博士）

## 客户需求
1. 润色投稿，1-3个月录用
2. 一区期刊
3. 担心找同行评审是否会泄露

## 沟通过程
### 阶段1：初次咨询
- 咨询产科合并神经系统疾病投一区或二区
- 询问润色投稿和快速见刊
- **销售澄清**：录用到见刊是期刊安排，无法干涉

### 阶段2：了解流程
- 客户询问1-3个月流程
- 担心同行评审是否会找到领导
- 销售解释：同行都是医学领域专家，不会泄露

### 阶段3：时间紧迫
- 客户说明：申博士需要1月份录用
- 销售强调：本月名额不多，需抓紧
- 用案例说明一区可以做到

## 成功原因
1. 精准把握客户时间紧迫性
2. 坦诚澄清"见刊"问题
3. 用案例证明实力
4. 名额策略促进成交

## 可复用的营销话术
1. **解释见刊**："录用到见刊是期刊和出版集团对接的，这个谁也没办法干涉"
2. **消除顾虑**："比彩票的概率还要低（泄露问题）"
3. **案例证明**："人家跟你们一样发一区，都发了两篇了"
4. **时间策略**："这个月已经没几个名额了，要做就要抓紧"
